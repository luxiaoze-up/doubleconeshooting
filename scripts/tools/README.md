# 工具脚本目录

本目录包含开发、维护和文档处理相关的临时工具脚本。

## 文档转换工具

- `convert_excel_to_md.py` - 将 Excel 文件转换为 Markdown 格式
- `convert_word_to_md.py` - 将 Word 文档转换为 Markdown 格式
- `convert_filtered_to_md.py` - 将过滤后的 PLC Tags Excel 文件转换为 Markdown

## 数据处理工具

- `filter_plc_tags.py` - 过滤 PLC Tags Excel 文件，删除备用及无意义的 Tag
- `dump_tango_db.py` - 导出 Tango 数据库配置信息

## 查看工具

- `view_markdown.py` - Markdown 文件查看器（Web 界面）

## 测试脚本

- `test_plc_communication.py` - PLC 通信测试脚本
- `test_image_api.py` - 图像 API 测试脚本

## 使用说明

这些脚本主要用于开发和维护工作，不是项目运行所必需的。如需使用，请参考各脚本的注释或文档。
